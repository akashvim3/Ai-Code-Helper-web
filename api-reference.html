API Overview

The CodeAssist AI API provides programmatic access to our AI-powered code completion and analysis features.

Base URL

https://api.codeassist.ai/v1

Quick Example

JavaScript 
const response = await fetch('https://api.codeassist.ai/v1/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    code: 'function calculateSum(',
    language: 'javascript',
    maxSuggestions: 3
  })
});

const data = await response.json();
console.log(data.completions);

Authentication

All API requests require authentication using an API key. Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

POST /completions

Generate AI-powered code completions based on the provided context.

Request Body

Parameter Type Required Description
code string Yes The code context to complete
language string Yes Programming language (e.g., javascript, python)
maxSuggestions integer No Maximum number of suggestions (default: 5)
