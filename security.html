Security

Your Security is Our Priority

Enterprise-grade security measures to protect your code and data

End-to-End Encryption

All data transmissions are encrypted using TLS 1.3 protocol. Your code is encrypted at rest using AES-256 encryption.

Privacy Protection

We never store your code permanently. Code snippets are processed in real-time and immediately discarded after generating suggestions.

SOC 2 Type II Certified

Independently audited and certified for security, availability, processing integrity, confidentiality, and privacy.

Regular Security Audits

Quarterly penetration testing and security audits by independent third-party security firms.

Multi-Factor Authentication

Protect your account with 2FA using authenticator apps, SMS, or hardware security keys.

Data Backup & Recovery

Automated daily backups with point-in-time recovery and 99.999999999% durability.

Security Best Practices

Use Strong Passwords

Create unique, complex passwords with at least 12 characters including uppercase, lowercase, numbers, and symbols.

Enable Two-Factor Authentication

Add an extra layer of security to your account by enabling 2FA in your account settings.

Rotate API Keys Regularly

Change your API keys every 90 days and immediately if you suspect any compromise.

Monitor Account Activity

Regularly review your account activity logs and report any suspicious behavior immediately.

Report a Security Vulnerability

If you discover a security vulnerability, please report it to our security team:

Email: security@codeassist.ai

PGP Key: Available on request

Bug Bounty: Rewards up to $10,000 for valid security issues
